Tag Description
<header> Header for a document or section.
<nav> Navigational links.
<main> Dominant content of the document.
<section> A thematic grouping of content.
<article> An independent, self-contained content.
<aside> Content aside from the page content.
<footer> Footer for a document or section.
<strong> Strongly emphasized, usually bold.
<em> Emphasized text, usually italics.
<code> Computer code - monospaced.
<mark> Marked or highlighted text.
<figure> Self-contained content such as an image, often with a caption.
<figcaption> A caption for a figure.
<div> Generic container for grouping elements (block).
<span> Inline container for text or other inline elements.
